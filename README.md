# Romantic Website Template

Невеликий відкритий шаблон романтичного сайту українською мовою. Його можна швидко налаштувати під себе та подарувати коханій людині як запрошення на побачення або просто милий сюрприз.

## Що всередині

- Статичний сайт без збірки: `HTML`, `CSS`, `JavaScript`
- Інтерактивна сцена з кнопками та GIF-анімаціями
- Фінальна романтична сцена із зірками та музикою
- Українські тексти за замовчуванням

## Структура

- `index.html` - основна сторінка та метатеги
- `public/css/style.css` - стилі
- `public/js/script.js` - логіка головної сцени
- `public/js/stars.js` - фінальна сцена зі зірками
- `public/images/` - GIF та інші зображення
- `public/audio/` - аудіофайли

## Як запустити локально

Спочатку склонуй репозиторій:

```powershell
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

Оскільки це звичайний статичний сайт, достатньо відкрити `index.html` у браузері.

## Як кастомізувати

Найчастіше достатньо змінити:

- текст вступу та запитання у `index.html`
- фрази кнопки `Ні` у `public/js/script.js`
- фінальні романтичні репліки у `public/js/stars.js`
- превʼю для поширення через `meta` та `og:*` теги у `index.html`

## Порада перед публікацією на сервер (для жоских програмістів)

Перед деплоєм варто оновити в `index.html`:

- `og:url` на реальну адресу сайту
- `og:image` на публічно доступне зображення для превʼю
- `title` і `description` під власний варіант подарунка
